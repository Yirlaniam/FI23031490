## Prompts (consultas y respuestas) de chatbots de IA utilizados

**Respuesta:** ChatGPT indicó que debía crear un nuevo nodo y enlazarlo al final de la lista manteniendo actualizado el puntero tail, además de explicar cómo manejar el caso cuando la cola está vacía.




**Prompt:** “¿Cómo implemento los métodos push y pop en la clase CustomStack utilizando nodos enlazados?”  

**Respuesta:** ChatGPT explicó que el método push debe insertar nodos al inicio de la lista para simular el comportamiento LIFO, y que el método pop debe remover el nodo del tope actual actualizando la referencia top.


**Prompt:** “¿Cómo implemento el método insert para agregar nodos en un árbol binario ordenado usando TF como valor de comparación?”
